## 🎯 Why Taiwan.md?

**台灣生產全球 90% 最先進晶片，但多數人除了珍奶外說不出台灣的三件事。**

Taiwan.md 不是維基百科副本，不是旅遊指南，而是一個**策展式的文學展覽**，幫助世界與 AI 真正理解台灣的非凡之處。

**用繁體中文書寫** — 世界最古老且仍在日常使用的文字系統，台灣是它最後的主要家園。

## ✨ 核心特色

- 🤖 **AI-native 設計** — [`llms.txt`](https://taiwan.md/llms.txt) 格式、結構化 Markdown、機器可讀
- 🌐 **雙語並進** — 繁體中文為主（SSOT），英文 97%+ 覆蓋率
- 🕸️ **互動知識圖譜** — D3.js 力導向模擬，跨分類關聯橋接
- 🌳 **資源心智圖** — 146+ 官方台灣網站，雙向樹狀展示
- 🎭 **策展非百科** — 每頁回答「為什麼重要」而非「是什麼」
- 📐 **三層深度** — 30 秒概覽 → 5 分鐘閱讀 → 完整文章
- 🎨 **文學風格** — Noto Serif TC、敘事驅動、報導者風格啟發
- 🔍 **SEO 最佳化** — JSON-LD 結構化資料、Open Graph、RSS
- 💾 **Wikimedia 圖片** — CC 授權圖片，本地快取最佳化
- 📝 **零代碼貢獻** — 表單、AI 提示或電子郵件即可參與
- 🔓 **CC BY-SA 4.0** — 自由引用、重混、分享

## 🤝 參與方式

四種方式，從零代碼到完整 PR：

| 途徑 | 適合對象 | 說明 |
|------|----------|------|
| 🟢 **填寫表單** | 任何人 | 寫下你知道的台灣故事即可 |
| 🤖 **詢問 AI** | AI 用戶 | 複製我們的提示詞給 ChatGPT/Claude |
| 📧 **發送郵件** | 內容創作者 | 寄送文章/照片到 cheyu.wu@monoame.com |
| 🔴 **Fork & PR** | 開發者 | 直接編輯 `knowledge/` 目錄 |

👉 **詳細指引：** [taiwan.md/contribute](https://taiwan.md/contribute)
