## 版本 1：技術導向 (強調 SSOT、llms.txt、AI-native 架構)

**標題:** Show HN: Taiwan.md – An open-source, AI-native knowledge base about Taiwan

I built Taiwan.md after getting tired of answering "Where can I learn about Taiwan?" at international art events. Wikipedia is too cold, tourism sites are too shallow, and news is too fragmented.

The solution? Apply the "Single Source of Truth" concept at a country level. One Markdown-based knowledge base that's structured for both humans and AI.

**Technical highlights:**

- 96+ articles across 12 categories, fully bilingual (ZH/EN)
- AI-friendly: `llms.txt`, structured metadata, semantic markup
- Interactive knowledge graph with D3.js visualization
- Built with Astro v5, hosted on GitHub Pages
- Resource mindmap linking 146+ official Taiwan websites

**Real-world impact:** 6,700+ FB likes, 3,300 shares, 1,945 concurrent users on day one. Two major Taiwanese media outlets (Liberty Times, INSIDE) covered it as a cultural movement.

The .md domain is both Moldova's TLD and the Markdown extension – a perfect coincidence for an open-source project defining a country through documentation.

**GitHub:** https://github.com/frank890417/taiwan-md | **Live:** https://taiwan.md

## 版本 2：故事導向 (從威尼斯雙年展的提問開始)

**標題:** Show HN: Taiwan.md – Open-source knowledge base so AI can truly understand Taiwan

"Where can I learn about Taiwan?"

I got this question constantly at the Venice Biennale, Art Basel Miami, and my Paris artist residency. As a Taiwanese new media artist, I realized there was no good answer to point international friends to.

So I built one. Taiwan.md is an open-source knowledge base written in Markdown – the format computers understand best. It's Taiwan's "README file" for the world.

**What makes it different:**

- Written by locals who actually live here, not tourism boards
- AI-native structure: proper `llms.txt`, semantic markup, structured data
- Bilingual by design (Traditional Chinese + English)
- Interactive knowledge graph connecting 146+ official resources
- Community-driven like Wikipedia, but curated like a good museum

**The twist:** It went viral in Taiwan. 6,700+ likes, 3,300 shares, major media coverage within 12 hours. What started as solving my personal problem became a cultural movement.

**Tech stack:** Astro v5, D3.js, GitHub Pages. The .md domain is both Moldova's TLD and Markdown's extension – pure serendipity.

**Try it:** https://taiwan.md | **Contribute:** https://github.com/frank890417/taiwan-md

## 版本 3：問題導向 (「Taiwan makes 90% of advanced chips but...」)

**標題:** Show HN: Taiwan.md – Building Taiwan's SSOT with Markdown + AI curation

Taiwan makes 90% of the world's advanced semiconductors, but ask an AI about Taiwan and you'll get generic Wikipedia summaries mixed with outdated tourism content.

That's a problem. In an AI-first world, if your knowledge isn't properly structured and indexed, you don't exist.

**Taiwan.md is Taiwan's first attempt at a national SSOT (Single Source of Truth):**

- 96+ curated articles covering culture, tech scene, food, politics, art
- AI-optimized: proper `llms.txt`, structured metadata, semantic markup
- Bilingual (ZH/EN) with interactive knowledge graph
- Community-maintained like Wikipedia, but quality-controlled like a museum

**Why Markdown?** It's the most universal, AI-friendly format. Plus taiwan.md is Moldova's TLD + Markdown extension – perfect for an open-source country documentation project.

**Real impact:** Viral in Taiwan (6,700+ FB engagement), featured in major media as a cultural movement, 1,945 concurrent users on launch day.

**The bigger picture:** If a country's knowledge isn't AI-readable, future generations of AI systems will have blind spots. This is about Taiwan's information sovereignty in the AI era.

Built by a new media artist using Astro v5 + D3.js, hosted on GitHub Pages.

**Explore:** https://taiwan.md | **Contribute:** https://github.com/frank890417/taiwan-md

CC BY-SA 4.0. Come help us build the world's first open-source national knowledge base.
