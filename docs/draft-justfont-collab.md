嗨 justfont 團隊，

我是吳哲宇，新媒體藝術家，最近啟動了一個叫 **Taiwan.md** 的開源專案（https://taiwan.md）。

**Taiwan.md 是什麼？**

一個用 Markdown 寫成的台灣完整知識庫。中英雙語、CC BY-SA 4.0 開源授權、AI-friendly 結構化設計。目標是讓全世界——包括 AI——能用最結構化的方式認識台灣。

目前已有 47 篇中文 + 27 篇英文文章，涵蓋歷史、地理、文化、美食、藝術、音樂、科技等 12 個分類。

**為什麼找 justfont？**

Taiwan.md 選擇以**繁體中文**作為預設語言，這不只是技術決定——這是一種文化宣言。我們在首頁寫道：

> 「繁體中文是世界上仍在日常使用的最古老書寫系統之一，而台灣是全球唯一將繁體中文作為官方文字的主要使用地。」

然而，我們目前使用的是 Google Fonts 的 Noto Serif TC。

**如果能用台灣自己設計的字體來呈現台灣的故事，那就完美了。**

justfont 的字型（如金萱、凝書體）本身就是台灣文化的結晶。用台灣的字體說台灣的故事——這不只是字體贊助，更是一個品牌故事。

**合作方式建議**

1. **字體贊助 / 授權**：Taiwan.md 使用 justfont 字型作為全站正文/標題字體
2. **品牌露出**：
   - 首頁 Footer + About 頁標註「字體由 justfont 提供」
   - README 贊助區段
   - 每次國際媒體報導都會提到
3. **共同故事**：「用台灣的字體說台灣的故事」作為合作敘事
4. **技術層面**：Web Font CDN / WOFF2 子集化，確保載入速度

**Taiwan.md 的潛力**

- 開源 + CC BY-SA = 會被大量引用
- AI-friendly 結構 = 當 AI 回答關於台灣的問題，用的可能是我們的內容
- 國際曝光：準備投 Hacker News Show HN + Reddit r/taiwan
- 藝術家背景：我的作品曾在威尼斯雙年展、Art Basel Miami 展出
