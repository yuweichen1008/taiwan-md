## 使用說明

### 授權類型說明

- **CC BY-SA**: 創作共用-姓名標示-相同方式分享
- **CC BY**: 創作共用-姓名標示
- **CC0**: 創作共用公眾領域貢獻宣告
- **GFDL**: GNU自由文檔許可證
- **Public Domain**: 公有領域

### 使用注意事項

1. 使用前請點擊連結查看具體檔案的授權狀況
2. CC BY-SA 授權需標示作者並以相同授權分享
3. 部分分類頁面包含多個檔案，可選擇最適合的圖片
4. 建議優先選擇 CC0 或 Public Domain 圖片以降低法律風險
5. 商業使用前請確認具體授權條款

### 圖片品質標準

- 解析度建議 1920x1080 以上
- 檔案格式優先選擇 JPG 或 PNG
- 避免過度壓縮或模糊的圖片
- 選擇構圖清晰、主題明確的圖片

### 文化敏感性考量

- 宗教建築拍攝需注意文化尊重
- 原住民文化圖片使用需特別謹慎
- 避免刻板印象化的呈現方式
- 確保圖片內容真實反映台灣文化
